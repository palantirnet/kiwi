Kiwi by Palantir.net

About

Kiwi is a PHP command line application that maps data from KE Software's EMu collection management system into the Apache Solr search server.

Install

  1. Download the source (via git or tarball.)
  2. Copy the contents of solr-conf to the conf directory of your Solr core.
  3. Create a configuration file.
  4. Run kiwi.php
  5. Profit!
