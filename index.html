kiwi by palantirnet

KEmu->Solr processing bridge

Kiwi is a PHP command line application that maps data from KE Software's Emu collection management system into the Apache Solr search server.

Dependencies

All open source dependencies are included in the source already. However, it does require installation of the Emu PHP library from KE software to connect to the Emu server.

Install

1) Download the source (via git or tarball.) 2) Download and install the Emu library. 3) Copy the contents of solr-conf to the conf directory of your Solr core. 4) Create a configuration file. (See docs for details.) 5) Run kiwi.php 6) Profit!
