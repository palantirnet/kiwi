Running Kiwi

To run Kiwi, you need only to run its PHP script from the command line (or a batch script). Change into the directory where you have Kiwi installed and run this command:

$ php ./kiwi.php

(You may also set the kiwi.php script to be executable, and then omit the "php" at the beginning.)

Running Kiwi without any command line switches will output brief help text. To process a specific configuration file, specify it with the --config parameter:

$ php ./kiwi.php --config=myconfig.xml

That command will execute the configuration file myconfig.xml, in the current directory, which conforms to the format specified by config-schema.rng and config-example.xml. See config-example.xml for further details on how to structure the configuration file.

To execute all configuration files in a particular directory, use the --directory parameter:

$ php ./kiwi.php --directory=config_files

That will iterate over all files in that directory and execute each of them in turn as a configuration file.

For additional command line options, run

$ php ./kiwi.php --help
